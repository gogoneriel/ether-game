# Valdemar Flag — Fountain Decoration

## What
Add a red banner/flag near the town fountain with the text "Valdemar" on it.

## Where
- Placed next to the fountain in Magnolia town square.
- Visible to all players walking through the plaza.

## How
- Red flag sprite (simple pixel-art cloth on a pole).
- "Valdemar" text rendered on the cloth.
- Static decoration — no interaction needed for v1.

## Future hooks
- Could become a template for player-planted name flags or guild banners later.

## Changelog
- v1 → v2: renamed flag text from "Fabiano" to "Valdemar" per owner request.
